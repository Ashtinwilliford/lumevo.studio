- Minimal, calm, premium, creator-focused
- No emojis
- Soft pastel yellow background
- Red accent color for buttons and key text
- Rounded corners
- Clean typography
- Spacious layout
- Feels like a mix of Canva, Notion, and Apple-style simplicity

Brand vibe:
- intentional
- elevated
- creative
- modern
- soft but confident

Core app flow:
1. Landing page
2. Dashboard
3. Create project
4. Project page
5. Upload page
6. Script page

Pages needed:

1. Landing Page
- Clean hero section

3. Create Project Page

- Textarea for content description

5. Upload Page

6. Script Page
