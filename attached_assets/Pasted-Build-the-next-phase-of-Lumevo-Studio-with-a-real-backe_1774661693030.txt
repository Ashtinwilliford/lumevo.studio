Build the next phase of Lumevo Studio with a real backend architecture, database schema, job pipeline, and app page structure.

This app is an AI social media manager that learns from what users upload and then creates content and edited videos in their style.

## Core Product Logic

The app should work like this:

1. User signs up
2. User uploads videos, images, captions, scripts, or ideas
3. System analyzes the uploads
4. System updates the user’s brand/personality profile
5. User chooses a target output:
   - caption
   - hook
   - post
   - video script
   - 30 second video
   - 1 minute video
   - 2 minute video
   - 3 minute video
6. System creates content in the user’s style
7. For video generation, system can:
   - mute original audio
   - keep partial audio if useful
   - add background music
   - generate voiceover
   - assemble clips/images into one finished video
8. Final content is saved to the user’s project history

This should be built as a real full-stack product, not a mockup.

# JOB QUEUE / WORKER PIPELINE

Do not render videos directly in the request cycle.

Set up async jobs and workers.

Create jobs for:

## upload_analysis_job
- reads uploaded assets
- extracts transcript and metadata
- stores analysis results
- triggers brand profile update

## brand_profile_update_job
- updates the user’s learning profile
- recalculates confidence score
- saves learned tone/style/personality

## content_generation_job
- creates captions, hooks, or scripts
- saves results in generated_contents

## creative_brief_job
- builds the structured plan for a video project
- creates creative_briefs record

## voiceover_generation_job
- sends script to ElevenLabs
- stores voiceover file and status

## music_selection_job
- picks a music track
- stores project music record

## render_video_job
- builds the timeline JSON
- renders vertical video
- stores final MP4
- updates project and render_jobs statuses

## notification_job
- notifies frontend when processing steps finish
- supports progress states

# SUBSCRIPTION FEATURE GATING

Implement 4 tiers:

## Free – Starter
- limited uploads
- limited generations
- no advanced brand memory
- no premium video generation

## Creator – Growth
- more generations
- basic personalization
- weekly content idea generation
- basic brand learning

## Pro – Brand Builder
- full brand personality profile
- stronger generation in their style
- multi-platform generation
- better project history
- advanced planning features

## Elite – AI Manager
- AI Video Creator + Manager
- best personalization
- longest video options
- premium voiceover tools
- advanced creative brief system
- strongest analytics
- highest limits

Add backend checks so premium features are blocked for lower tiers.

- use uploaded videos and images

- add one background music track

- save to project history

Keep 2 minute and 3 minute support structured but optional for first implementation.
